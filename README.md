# pyqgis-cookbook

PyQGIS cookbook in markdown and Jupyter notebook formats

This repository contains the PyQGIS cookbook in markdown and Jupyter notebook formats. They are generated from the RST source files in the `rst` directory, which are copied from the [QGIS documentation repository](https://github.com/qgis/QGIS-Documentation/tree/master/docs/pyqgis_developer_cookbook). Credits to the QGIS documentation team for the original content.
